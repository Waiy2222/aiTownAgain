#### Test-003: 活跃对话数实时更新

- **类型**：手动测试 / 实时监控
- **测试数据**：等待 NPC 自动触发对话（约 30-60 秒内）
- **测试方案**：
  1. 观察 Dashboard 上"活跃对话数"指标
  2. 当有 NPC 开始对话时，记录数字变化
  3. 当对话结束时，记录数字变化
  4. 同时观察 PixiJS 层 NPC 是否在对话

- **预期结果**：
  - NPC 开始对话 → 活跃对话数 +1
  - 对话结束 → 活跃对话数 -1
  - 数字变化带过渡动画
